# NEW-BANK-API

## Endpoints

* GET /v1/user/ - Get all users
* GET /v1/user/:id - Get by id
* POST /v1/user/ - Create user
* PATCH /v1/user/:id - Update user by id
* DELETE /v1/user/:id - Delete user by id

## Documentation: 

* GET /docs/ 

## Technologies

* Golang
